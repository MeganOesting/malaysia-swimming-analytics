What are Malaysia Age Points?

Malaysia Age Points (MAP) is an age-adjusted scoring system that allows fair comparison of swimming performances across different ages, events, and strokes. MAP is based on the same principles as Hy-Tek Power Points used by USA Swimming.

Key Characteristics:

  • Age-Adjusted: A 500-point swim at age 12 represents the same relative quality as a 500-point swim at age 16
  • Single-Age Tables: Each age (12, 13, 14, 15, 16, 17, 18) has its own base time table
  • Development Focus: Designed to identify and track developing swimmers, not just absolute speed
  • Cross-Event Comparison: Compare a swimmer's 100 Fly against their 200 Free performance

The Formula

MAP uses the same cubic curve formula as AQUA points:

MAP = 1000 x (B / T)^3
VariableDescriptionKey Difference from AQUA
MAPPoints awarded-
BBase time (in seconds)Age-specific base time (USA #100 All-Time)

The only difference: MAP uses age-specific base times, while AQUA uses a single base time (world record) for all ages.

Base Times: USA All-Time Top 100

MAP base times are derived from the 100th fastest time ever swum by an American swimmer at each specific age.

Why USA Data?

  • Largest Dataset: USA Swimming has the world's largest database of age-group swimming times
  • Consistent Tracking: Times database goes back decades with reliable age verification
  • Depth of Competition: Ensures statistically meaningful benchmarks at every age

Why Use Age Points?

Age-adjusted points (like MAP and Hy-Tek Power Points) serve several important purposes:

1. Identify Developing Talent

  • A young swimmer with high MAP points shows strong potential relative to peers
  • Helps coaches spot talent that might be overlooked based on absolute times alone

2. Track Progress Fairly

  • Compare a swimmer's 14-year-old self to their 16-year-old self using the same scale
  • Determine if improvement is keeping pace with expected development

3. Cross-Event Analysis

  • Is a swimmer stronger in sprint or distance events?
  • Which stroke shows the most potential?
  • Compare 100 Fly (580 MAP) vs 200 Free (620 MAP) to guide training focus

4. Team & Program Evaluation

  • Squad competitions using total MAP points create fair multi-age competitions
  • Track program development across age groups over time
  • Award recognition based on relative achievement, not just fastest times

5. Meet Qualification

  • Some programs use minimum MAP points for meet entry
  • Ensures appropriate competition level regardless of event selection
